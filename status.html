Ce que signifient les chiffres

  • visites (local) — nombre total de chargements de page enregistrés dans le localStorage de votre navigateur. Chaque visiteur ne voit que ses propres statistiques.
  • aujourd'hui — visites depuis minuit (votre heure locale).
  • temps moyen — temps moyen passé sur cette page durant vos sessions.
  • pages — nombre de chemins de pages uniques que vous avez visités sur ce site.
  • clics enregistrés — total des clics sur les liens, boutons et autres éléments enregistrés depuis votre navigateur.
  • pages les plus visitées — les URL que vous avez le plus consultées, triées par nombre de vues avec un graphique à barres.
  • éléments les plus cliqués — les boutons/liens avec lesquels vous avez le plus interagi.
  • visites récentes — journal chronologique de vos pages vues avec la source (référent).

Toutes ces données vivent uniquement dans le localStorage de votre navigateur. Elles ne quittent jamais votre machine. Vous pouvez les effacer à tout moment avec le bouton « clear local ».

Token GitHub — à quoi ça sert

Ce champ stocke un token d'accès personnel GitHub dans votre localStorage afin qu'un futur script puisse appeler l'API GitHub (ex : récupérer les infos du repo, les étoiles, les commits, les issues). Il n'est utilisé par rien pour le moment — c'est juste un endroit sûr pour le conserver.

Le token n'est jamais envoyé ailleurs sauf si vous construisez explicitement une fonctionnalité qui l'utilise. Il reste dans votre navigateur, sur votre machine.

Quel type de token utiliser

  • Allez sur github.com/settings/tokens
  • Générez un token fin (fine-grained PAT) ou classique
  • Pour des données publiques en lecture seule, choisissez le scope public_repo (ou metadata:read)
  • Sans expiration est acceptable pour cette page utilitaire personnelle. Pour plus de sécurité, utilisez un token fin limité à ce repo uniquement.
  • Collez-le dans le champ ci-dessus et cliquez sur save — il est stocké dans votre localStorage.

⚠️ Cette page est en noindex, mais toute personne ayant l'URL peut l'ouvrir. Traitez votre token en conséquence.

Données serveur

La section « server-side intelligence » affiche vos données visiteur (IP, pays, ville, navigateur, langue), les infos de déploiement (environnement, région) et les métadonnées git (SHA du commit, branche, message, repo). Ces données proviennent des en-têtes de requête Vercel — rien n'est stocké côté serveur.

Qué significan los números

  • visitas (local) — total de cargas de página registradas en el localStorage de tu navegador. Cada visitante solo ve sus propias estadísticas.
  • hoy — visitas desde la medianoche (tu hora local).
  • tiempo medio — tiempo promedio en esta página durante tus sesiones.
  • páginas — rutas de página únicas que has visitado en este sitio.
  • clics registrados — total de clics en enlaces, botones y otros elementos registrados desde tu navegador.
  • páginas más visitadas — las URL que más has visto, ordenadas por recuento con gráfico de barras.
  • elementos más clicados — los botones/enlaces con los que más interactuaste.
  • visitas recientes — registro cronológico de tus páginas vistas con la fuente de referencia.

Todos estos datos viven solo en el localStorage de tu navegador. Nunca salen de tu máquina. Puedes borrarlos en cualquier momento con el botón « clear local ».

Token de GitHub — para qué sirve

Este campo almacena un token de acceso personal de GitHub en tu localStorage para que un futuro script pueda llamar a la API de GitHub (ej : obtener info del repo, estrellas, commits, issues). No se usa para nada aún — es solo un lugar seguro para tenerlo.

El token nunca se envía a ningún lado a menos que construyas explícitamente una función que lo use. Permanece en tu navegador, en tu máquina.

Qué tipo de token usar

  • Ve a github.com/settings/tokens
  • Genera un token fino (fine-grained PAT) o clásico
  • Para datos públicos de solo lectura, selecciona el alcance public_repo (o metadata:read)
  • Sin expiración está bien para esta página personal. Para mayor seguridad, usa un token fino limitado solo a este repo.
  • Pégalo en el campo de arriba y haz clic en save — se guarda en tu localStorage.

⚠️ Esta página está en noindex, pero cualquiera con la URL puede abrirla. Trata tu token en consecuencia.

Inteligencia del servidor

La sección « server-side intelligence » muestra tus datos de visitante (IP, país, ciudad, navegador, idioma), información de despliegue (entorno, región) y metadatos de git (SHA del commit, rama, mensaje, repo). Estos datos vienen de las cabeceras de solicitud de Vercel — no se almacena nada en el servidor.

ماذا تعني الأرقام

  • الزيارات (محلي) — إجمالي مرات تحميل الصفحة المسجلة في localStorage في متصفحك. كل زائر يرى إحصائياته الخاصة فقط.
  • اليوم — الزيارات منذ منتصف الليل (حسب توقيتك المحلي).
  • متوسط الوقت — متوسط الوقت الذي قضيته في هذه الصفحة خلال جلساتك.
  • الصفحات — عدد مسارات الصفحات الفريدة التي زرتها في هذا الموقع.
  • النقرات المسجلة — إجمالي النقرات على الروابط والأزرار والعناصر الأخرى المسجلة من متصفحك.
  • الصفحات الأكثر زيارة — عناوين URL التي شاهدتها أكثر، مرتبة حسب العدد مع رسم بياني.
  • العناصر الأكثر نقراً — الأزرار/الروابط التي تفاعلت معها أكثر.
  • الزيارات الأخيرة — سجل زمني لصفحاتك التي شاهدتها مع المصدر (المرجع).

جميع هذه البيانات تعيش فقط في localStorage في متصفحك. لا تغادر جهازك أبداً. يمكنك مسحها في أي وقت بزر « clear local ».

رمز GitHub — ما الغرض منه

هذا الحقل يخزن رمز وصول شخصي من GitHub في localStorage الخاص بك ليتمكن سكريبت مستقبلي من استدعاء GitHub API (مثلاً: جلب معلومات المستودع، النجوم، الالتزامات، المشاكل). لا يُستخدم لأي شيء بعد — إنه مجرد مكان آمن للاحتفاظ به.

الرمز لا يُرسل إلى أي مكان أبداً ما لم تقم ببناء وظيفة تستخدمه بشكل صريح. يبقى في متصفحك، على جهازك.

أي نوع من الرموز استخدام

  • اذهب إلى github.com/settings/tokens
  • قم بتوليد رمز دقيق (fine-grained PAT) أو كلاسيكي
  • للبيانات العامة للقراءة فقط، اختر نطاق public_repo (أو metadata:read)
  • بدون انتهاء صلاحية جيد لهذه الصفحة الشخصية. لمزيد من الأمان، استخدم رمزاً دقيقاً مقتصراً على هذا المستودع فقط.
  • الصقه في الحقل أعلاه وانقر save — سيتم تخزينه في localStorage الخاص بك.

⚠️ هذه الصفحة في noindex، لكن أي شخص لديه الرابط يمكنه فتحها. تعامل مع رمزك وفقاً لذلك.
